# EZ4H
Allow Minecraft:JE(1.12.2) connect to Minecraft:BE(1.16.200) servers!  
## Features
- [x] Login  
- [x] Chat  
- [x] Command  
- [X] Xbox Auth  
- [X] Chunks  
- [X] Move  
- [X] Players  
- [X] Entitys  
- [X] Block Entitys  
- [X] Item Entitys  
- [X] Entity Metadata  
- [X] Entity Interact  
- [X] Block Interact  
- [X] Titles and other Messages  
- [X] Inventory Action  
- [ ] UIes  
- [ ] Particles  
- [X] Form UI  
- [X] Level Events  
- [ ] Sounds
